# lab-mqtt-grupoG
##GrupoG
No grupo G, composto por Gabriel, Luan e Victor, todos cursando técnico em eletrônica, será apresentado um projeto no qual visa a qualidade de insumos, bem como a segurança e saúde de quem os consome, através de monitoramento em tempo real da temperatura de câmeras frias e frigorificos, onde caso a temperatura saiá da faixa ideal, um alerta visual será acionado para verificação e regulagem imediata.
