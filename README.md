# lab-mqtt-grupoG
## GrupoG
## Projeto: Monitoramento de temperatura em câmaras frias/frigorificos.
Integrantes: Gabriel Balduino dos Santos, Luan César Pereira e Victor Augusto Barcarollo. 
Descrição do projeto:
Sistema de monitoramento de temperatura para câmara frias utilizando ESP32, MQTT e VPS, com alertas em tempo real para variações fora da faixa 1°C a 5°C.
